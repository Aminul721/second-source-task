What's the closure

A closure happens when a function is defined inside another function and retains access to the outer function’s variables, even after the outer function has returned.

function outerFunction() { let count = 0; function innerFunction() { count++; console.log(count); } return innerFunction; } const counter = outerFunction(); counter(); // Output: 1 counter(); // Output: 2 counter(); // Output: 3
